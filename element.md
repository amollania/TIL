Java Starting
============

真偽値
--------------
'''java
boolean bool = true; //ダブルクォーテーションはつけない
'''

比較演算子
---------------
'''java
6+2 ==5 // false
6+2 !=5 //true
6/3 ==2 //true
6/3 !=2 //false
'''

真偽値を出力
----------
'''java
System.out.println(true);   // true
System.out.println(false);  // false
System.out.println(6+2==5); // false
System.out.println(6+2!=5); // true
'''
